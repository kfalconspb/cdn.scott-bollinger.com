The Purpose of Network Analysis

Network analysis often called sniffing or protocol analysis, is the act of listening to communications traversing the network and breaking down the traffic into its basic components to be examined. This allows the analysis to be filtered and focused. Protocol analysis is useful in baselining normal traffic, identifying security issues, or misconfigured/malfunctioning equipment.
