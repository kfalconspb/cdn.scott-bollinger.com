Create VMware ESXi USB Install Media

Posted by Scott Bollinger / @kfalconspb on August 05, 2018 #Virtualization

Something Physical

It's not often I install anything from physical media anymore, but it still has it's uses. This USB stick, will be used to install ESXi, on my Intel NUC's, and the Mac Mini, in am upcoming post. I'll explore PXE boot in the future, as well

Thanks,

Scott Bollinger / @kfalconspb

Prep

Download the ESXi ISO from the VMware Download Website. The ISO needs to be written to the USB, and be made bootable. There are many applications which can do this for you, in this example we'll be using UNetbootin.

Download the appropriate version for the OS, or in this case we'll be using apt-get to install it.

Open your terminal console of choice, and type the following to add the apt repository for it: sudo add-apt-repository ppa:gezakovacs/ppa

Type: sudo apt-get update

Then type: apt-get install unetbootin

Type Y, to install

Installing

From the Ubuntu desktop. select activities, and search for unetbootin, and click to open.

UNetbootin gives the option to select, and download a distribution to install, or an ISO can be chosen.

Select Diskimage, navigate to, and select the VMware ESXi downloaded from earlier. Click OK to begin.

Extracting files to be written to USB.

Installing bootloader.

Click Exit, when the installation is complete.

Viewing the USB in a file manager, the VMware install files can be seen.
