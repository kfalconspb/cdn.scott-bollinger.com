Mount the ISO, and Install

Prepare a Windows 2016 VM, patched current, network configured, joined to the domain.

Download, and mount, the SQL ISO from Microsoft Downloads

Double click on the Mounted Drive, and begin the installation.

The initial Planning section has installation documentation, a system configuration checker, as well as an Installation Advisor to identify possible issues that need to be resolved.

Click on the Installation section on the left. Select the installation type. This install we are selecting a stand-alone installation.

Select Evaluation, or enter your license key. Click Next.

Accept the EULA, and click Next.

Select Check for Updates, and click Next.

It's scanning for updates.

Click Next.

Select the features to be installed, and click Next.

Name the instance, or accept the default, and click Next.
