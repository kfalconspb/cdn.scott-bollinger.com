VMware License Entitlement vSphere 6 Product Features

Licensing

[2]

The first vSphere 6 product feature I'm posting about is License Entitlement. Meaning what do you get for each license.

VMware vSphere is licensed per physical processor on the host. [1] Each physical processor must have a vSphere 6 processor license key in order to run vSphere. [2 pg 4]

There is no limit to the number of physical cores or memory per licensed physical processor.
